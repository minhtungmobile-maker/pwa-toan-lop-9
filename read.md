## 🚀 Cách triển khai

### 1. Deploy lên GitHub Pages
1. Tạo repo public tên: `pwa-toan-lop-9`
2. Upload toàn bộ thư mục vào repo
3. Vào **Settings → Pages**:
   - Branch: `main`
   - Folder: `/ (root)`
4. Truy cập:  
   👉 `https://[tên-bạn].github.io/pwa-toan-lop-9/`

### 2. Cài như app trên điện thoại (Android)
1. Mở Chrome → truy cập link trên
2. Nhấn **⋮ (menu)** → **“Thêm vào màn hình chính”**
3. App **“Toán Lớp 9”** sẽ xuất hiện như Zalo, Facebook

→ **Dùng được ngay cả khi không có mạng!**
